Essential Commands

Simple slash commands for complete music control

/play [song]

Play a song or add it to the queue

/pause

Pause the current track

/skip

Skip to the next song

/stop

Stop playback and clear queue

/queue

View the current queue

/nowplaying

See what's currently playing

/shuffle

Shuffle the queue

/loop [mode]

Loop song or queue

/volume [0-200]

Adjust volume (Premium)

/filters

Apply audio filters

/247

Enable 24/7 mode (Premium)

/lyrics

Display song lyrics (Coming Soon)

/premium

View premium features

/recommend

Get AI recommendations

/stats

View bot statistics
