Documentation & Troubleshooting

Everything you need to know about using Stan's Beat Bot

🚀 Getting Started

  1. Invite the bot to your Discord server using the invite link
  2. Join a voice channel in your server
  3. Use /play [song name] to start playing music
  4. The bot will join your voice channel and start playing!

⚠️ Troubleshooting

Bot not joining voice channel?

  • Make sure you're in a voice channel first
  • Check bot permissions (needs "Connect" and "Speak" permissions)
  • Try using /stop then /play again

No sound playing?

  • Check your Discord voice settings (User Settings → Voice & Video)
  • Make sure bot isn't muted in the voice channel
  • Try adjusting volume with /volume 100 (Premium only)

Commands not working?

  • Make sure you're using slash commands (type / to see available commands)
  • Bot needs "Use Application Commands" permission
  • Try kicking and re-inviting the bot if commands don't appear

Bot keeps disconnecting?

  • Enable 24/7 mode with /247 (Premium only)
  • Bot auto-disconnects after 60 seconds if no one is in the voice channel
  • Check your server's voice region settings
