All Commands

Full slash command reference for Stan's Beat Bot

▶ Playback

/play [song]

Play a song or add it to the queue

/playnow [song]

Immediately play a song, interrupting the queue

/playtop [song]

Add a song to the top of the queue

/playskip [song]

Add a song and instantly skip to it

/pause

Pause the current track

/resume

Resume a paused track

/skip

Skip to the next song

/stop

Stop playback and clear the queue

📋 Queue

/queue

View the current queue

/nowplaying

See what's currently playing

/shuffle

Shuffle the queue

/loop [mode]

Loop the current song or entire queue

/move [from] [to]

Move a song to a different queue position

/remove [position]

Remove a song from the queue

/clear

Clear the entire queue

/seek [time]

Seek to a position in the current track

🔍 Search & Info

/search [query]

Search for songs and pick from results

/lyrics

Display lyrics for the current song

/stats

View bot statistics and server usage

/recommend

Get AI-powered song recommendations

💎 Premium

/volume [0-200]

Adjust playback volume · Premium

/filters

Apply audio filters (bass boost, nightcore, etc.) · Premium

/247

Enable 24/7 stay-in-voice mode · Premium

/premium

View and manage your premium subscription
